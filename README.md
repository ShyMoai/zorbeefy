# Cenafy

A Google Chrome Extension has a 1/100 chance of injecting the Zorbeez ad by billy mays whenever you load up a new tab.
HOW TO ADD TO CHROME : 
1. Download this repository as a .zip folder
2. Extract the zip folder
3. Go to chrome://extensions/
4. Enable Developper Mode
5. Click "Load unpacked" and select "zorbeefy-master"
6. Select the folder inside zorbeefy-master (should be zorbeefy-master too)
7. You all set ! Enjoy your Billy Mays !
